# AI Deep Research Generator v11 ⚡

A high-performance, long-form research document generator that leverages massive AI inference speeds to build multi-section papers in minutes. This version is a **Progressive Web App (PWA)**, designed for mobile installation and seamless desktop use.

## 🌟 Key Features
* **Multi-Engine Request Routing:** Intelligently rotates requests between up to 4 Groq keys and 2 Cerebras keys to maximize throughput and bypass individual rate limits.
* **PWA Enabled:** Install directly to your home screen on iOS, Android, or Desktop for an app-like experience.
* **Robust Document Assembly:** Generates documents from 10 to 300+ pages with rolling memory for cohesive transitions between sections.
* **Pro Exports:** Save your finished research as a formatted PDF, Microsoft Word (.doc), or HTML file.
* **Token Analytics:** Real-time tracking of prompt and completion tokens used per API key.

## 🚀 Setup & Installation
1. **GitHub Pages:** Access the live app via your GitHub Pages URL.
2. **PWA Install:** In your mobile browser, select "Add to Home Screen" or click the "Install App" button in the header.
3. **API Keys:** Obtain free API keys from:
   - [Groq Cloud Console](https://console.groq.com/)
   - [Cerebras Cloud](https://cloud.cerebras.ai/)
